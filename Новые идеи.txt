Идеи, взятые из CryptoIA (и как их применить)
============================================

1) Advanced TA Agent — вычисление свинг‑структуры/Фибоначчи → даёт контекст тренда и уровни для стопов/тейков.
2) Chart Vision Agent — многотаймфреймная визуальная оценка (эвристики + мультимодальная LLM опционально) → агрегировать таймфреймы.
3) Cognitive Architect — мета‑агент для улучшения промптов/параметров по логам → у нас как Research/Architect в sandbox.
4) Context Builder — сбор всех источников в единый контекст (режим/новости/он‑чейн/TA) → рабочая память для объяснимости.
5) Pattern Discovery — поиск больших движений и генерация JSON‑паттернов → pipeline кандидатов стратегий.
6) Red Team — стресс‑сценарии и анти‑кейсы → устойчивость risk‑политик/стратегий перед промо.
7) Self‑Critique — независимый контр‑анализ решений → понижение доверия/объёма при рисковых сигналах.
8) Technical Synthesis — агрегирование сигналов с весами и нормировкой → итоговый bias для портфеля.
9) Memory Guardian — база «уроков»/ошибок/советов (embeddings + поиск) → предотвращение повторения ошибок.
10) Investment Arbiter — финальный арбитр вероятностей/весов стратегий с ограничением доверия → сглаживает решения.
